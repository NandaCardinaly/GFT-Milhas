Spring Spring Tool Suite Thymeleaf WebJars Hibernate Bootstrap 4
Spring Utilizamos o Spring Boot com os seguintes starters: Spring MVC, Spring Security, Spring Validation, Spring Dev Tools e Spring Data JPA.
Spring Tools 4 para Eclipse Plugin do ambiente de desenvolvimento Eclipse que é customizado para trabalhar com aplicativos Spring. Ele fornece, pronto para uso, o ambiente para implementar, depurar, executar e implantar e incluir algumas integrações.
Thymeleaf O Thymeleaf é um engine para Java usado como alternativa ao JSP, as páginas são escritas em HTML5 e já tem uma boa integração com Spring, quando usamos o Thymeleaf ficamos livres de usar o JSTL nas nossas páginas, pois o mesmo já nos oferece recursos para criação de loops, estruturas condicionais entre outras funções.
WebJars WebJars são bibliotecas da Web do lado do cliente empacotadas em arquivos JAR (Java Archive). Utilizamos principalmente para personalizar nossa aplicação, incluindo a depedência do open-iconic.
Hibernate ORM Usamos o hibernate juntamente com JPA, para a criação dos métodos de CRUD, (Create, Read, Update, Delete).
Bootstrap utilizamos o bootstrap para desenvolver alguns componentes de interface junto com o HTML e CSS com a finalidade de melhorar a experiência do usuário.
